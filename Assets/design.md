# 卡牌追踪器设计文档

## 设计1

### 主页面

#### 玩家信息
- **角色头像（含皮肤）**
- **玩家用户名**
- **上一回合状态**：
  - 命元
  - 生命上限
  - 修为
  - 体魄
- **追踪中的手牌**
- **出现过的手牌**
- **等级**：
  - 名称
  - 境界（通过边框颜色显示）

#### 选择展示顺序
- 境界优先
- 门派优先
- 副职优先
- 机缘优先
- 等级优先
